## 📌 실행 방법

1. git clone을 한다.

2. repository 로 폴더 이동

```
cd javascript-lotto
```

3. step2의 브런치로 이동

```
git checkout step2
```

4. dependency 다운로드

```
yarn install
```

### step1

- step1 프로젝트 시작

```
yarn start-step1
```

- step2 프로젝트 시작
